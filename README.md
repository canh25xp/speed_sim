# Speed sensor simulation and data analaysis Mini Project

## How to compile

### Prerequisites

git, cmake, g++

#### Windows

```pwsh
winget install Git.Git
winget install Kitware.CMake
```

#### Ubuntu/Debian

```bash
sudo apt install git cmake g++
```

### Build

```sh
git clone https://github.com/canh25xp/speed_sim
mkdir build
cd build
cmake ..
cmake --build .
```
